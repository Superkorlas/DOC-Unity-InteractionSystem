**Namespace:** `FireSoftworks.Interaction`  
**Type:** Struct  

Represents the result of an executed interaction action.
Contains a success flag and an optional message describing the outcome.
This struct is serializable for logging and debugging purposes.

```csharp
[Serializable]
public struct InteractionResult
```
