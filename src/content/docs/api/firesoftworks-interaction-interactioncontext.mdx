**Namespace:** `FireSoftworks.Interaction`  
**Type:** Struct  

Immutable data structure passed to action validators and execution methods.
Contains contextual information about the interaction (interactor, target, environment).
This struct is read-only to ensure consistency across the interaction pipeline.

```csharp
public readonly struct InteractionContext
```
