Base class for hybrid interaction actions that combine a ScriptableObject (InteractableActionSO)
for shared logic/configuration with per-instance state in a MonoBehaviour.
Inherit from this class to easily create actions that need both SO-driven logic and unique state per interactable.

```csharp
public class InteractableActionSOWrapper : InteractableAction, IInteractableAction
```

## Methods

### `Awake()`

